家庭内网升级

发表于 2020-11-16 分类于 synology 阅读次数： 本文字数： 971 阅读时长 ≈ 4 分钟

之前用的小米路由3，小两房勉强够用。

搬到新家之后就明显不给力了。

户型图

新家的户型：

交付时的布线：

房间 网口
弱电箱-客厅前墙（电视） 网线 + 有线电视同轴线
弱电箱-客厅后墙 网线 + 电话线
弱电箱-主卧 网线 + 电话线
弱电箱-次卧 网线 + 电话线
弱电箱-书房 网线 + 电话线
弱电箱-卫生间 电话线
  • 网线和有线电视/电话线都是两根线同走一个穿线管
  • 电话线其实也是8芯网线，只不过接口是电话接口

1. 网络方案 V1

1.1 网络拓扑

小米路由器3放在电视墙那里，WAN口接弱电箱的光猫，LAN口接电视。

书房的网线接光猫。

缺点:

  • 书房电脑和电视、无线设备不在一个网段，
  • 光猫比较拉跨，时不时会死机。

于是，就考虑把电视墙那里的有线电视同轴线换成网线，这样就可以从小米路由器的LAN口接回弱电箱了。

2. 网络方案 V2

2.1 换线

起初比较担心不好穿线，犹豫了好久。

后来试了一下把网线固定死，只拔有线电视同轴线。

嗯，可以拽得动，还算比较轻松。那就开使换线。

网线哪里找？直接把去主卧的那根电话线抽了出来：）

用电工胶带把有线和网线搭一起大约30cm，缠好。

顺利拽出来了。

2.2 压头

借了一套压线钳、信号检测器，顺利整好。

终于解决了电视墙回弱电箱的问题。

2.3 弱电箱中转

弱电箱里随便用了一个网线延长器把回去的那个网线和书房的网线对接即可。

绿联: 网线转接头对接头 RJ45网线延长器

2.4 群晖 NAS

买了群晖 218+ 之后，有线网口也不够了。

一开始把 NAS 放到电视下面，但是没几天进风口就全是灰尘。

群晖噪音也很大，晚上很明显。

因此，又在弱电箱里加了一个 TP-LINK 的交换机（5口千兆）。

把群晖搬到了弱电箱旁边的柜子里。

啊呦，，，柜子底板是悬空的，共振起来噪音更大，简直不能忍。

在群晖底下垫了个鼠标垫，好了一点。

2.5 换路由器

无线信号其实也成问题，路由器放在电视墙那边，信号都被墙挡住了。

  • WIFI 5G 信号只有客厅有。
  • WIFI 2G 主卧室信号只有一半，卫生间和书房的信号关上门就经常中断了。

后来换了个小米 AX3600，块头超大，形状也忒怪异，太占地方。

好的是WIFI 信号的确比原来好多了，不再出现网络中断的现象。

但书房角落还是不是满格。。。

2.6 网络拓扑

3. 网络方案 V3

后来研究了一段 MESH 组网，还是放弃了。

贵且不说，无线信号再好也抵不上一根破5类网线。

最后觉得再折腾一次，玩个大的，这次把全屋的网线全换两根的。

  • 路由器放到客厅后墙沙发那一侧，直接正对过道，这样信号就不会受墙体阻挡了。
  • NAS 搬到书房，解决噪音的问题。
  • 有线网口都改为双网口：弱电箱-电视墙、弱电箱-客厅后墙、弱电箱-书房

3.1 材料、工具准备

  • 山泽(SAMZHE)压线钳
  • 信号测试器
  • 水晶头，六类， 50个
  • 网线， 六类， 50m×2卷
  • 双网口面板， 86型，千兆双口

网线六类足够了，基本能满足播放 NAS 影音的需求。

3.2 换线

换线都是体力活，略。

3.3 最终网络拓扑
