台达PLC处理模拟量信号

发表于 2020-09-13 分类于 mechatronics 阅读次数：

1. 硬件

  • 1.1 台达PLC

    • 型号： DVP-40EH00T2

  • 1.2 信号模块

    • 型号： DVP-04AD-H2

  • 1.3 参考文献

    • 台达PLC手册： DELTA_IA-PLC_DVP-EH3_I_TSE_20201022.pdf
    • 台达特殊模块手册： DVP-PLC应用技术手册（特殊模块篇）(326P).pdf

2. 接线

以两线制 4-20mA 信号为例，

  • 仪表正极接PLC的24V+，仪表负极接I+输入端子；
  • 模块的24V端接PLC的24V+，模块的com端接PLC的24V-

这样可以形成回路。

注：DVP-04AD模块测量电流信号时， V+ 及 I+ 端子需要短接

3. 程序

LD M1002
TO K0 K1 H3 K1       # H3 代表电流信号
LD M1002
TO K0 K2 K10 K1      # K10 代表平均次数 

LD M1000
FROM K0 K6 D40 K1    # 平均信号 -> D40
LD M1000
FROM K0 K12 D50 K1   # 实时信号 -> D50
LD M1000
DIV D50 K200 D0      # 电流范围0-20mA, D50的数值范围为K0-K4000； 缩小200倍后存入 D0
LD M1000
CMP D50 K1600 M0     # 比较命令，将比较结果存入 M0

...

详见技术手册。

另：

  • 输入输出点数不多的情况，可直接选用 EX 系列的PLC，PLC上自带模拟量的输入。
  • 如：DVP-20EX200R
