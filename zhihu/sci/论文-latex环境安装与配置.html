latex зҺҜеўғе®үиЈ…дёҺй…ҚзҪ®

title: latex зҺҜеўғе®үиЈ…дёҺй…ҚзҪ®
date: 2019-10-06 19:20:03
tags: latex
categories: sci

еңЁ win10 дёӢе°қиҜ•иҝҮзҡ„ latex зҺҜеўғжңүд»ҘдёӢдёүз§Қпјҡ

  • texlive + texstudio
  • ctex (йӣҶжҲҗ miktex + winedtпјҢ д»ҘеҸҠдёӯж–Үж”ҜжҢҒ)
  • miktex + texstudio / пјҲvscode + latex workshopпјү

1. texlive + texstudio

иҝҷдёӘжҳҜжңҖеёёз”Ёзҡ„й…ҚзҪ®ж–№жі•пјҢе»әи®®ж–°жүӢд»ҺиҝҷдёӘејҖе§ӢдҪҝз”ЁгҖӮ
дјҳзӮ№пјҡеҹәжң¬йҖӮз”ЁдәҺжүҖжңүзҡ„еңәжҷҜпјҲдёӯж–ҮйҷӨеӨ–пјүпјҢдёҖеҠіж°ёйҖёпјӣ
зјәзӮ№пјҡй»ҳи®ӨдёҚж”ҜжҢҒдёӯж–ҮпјҢе®үиЈ…еҢ…е·ЁеӨ§гҖӮ

  • texlive е®үиЈ…пјҡ еҸҜд»ҘеңЁзәҝе®үиЈ…пјҢд№ҹеҸҜд»ҘзҰ»зәҝе®үиЈ…пјҲжҺЁиҚҗпјҢе®үиЈ…еҢ…3.2GпјүпјҢе®үиЈ…еҗҺ6Gзҡ„з©әй—ҙгҖӮ

    • жҺЁиҚҗжё…еҚҺеӨ§еӯҰжәҗпјҡ https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/

  • texstudio е®үиЈ…пјҡдёӢиҪҪе®үиЈ…еҢ…пјҲ~89MпјүпјҢжӯЈеёёзҡ„е®үиЈ…ж–№жі•еҚіеҸҜгҖӮ

    • е®ҳзҪ‘дёӢиҪҪпјҡ http://texstudio.sourceforge.net/
    • Scoop е®үиЈ…пјҡ scoop install texstudio

  • texstudio й»ҳи®Өзҡ„й…ҚзҪ®е°ұеҹәжң¬еӨҹз”ЁпјҢдёҚиҰҒд№ұж”№гҖӮ

    • дёӯж–ҮпјҢеӯ—дҪ“пјҢеӯ—еҸ·
    • й«ҳзә§й…ҚзҪ®пјҡиЎҢеҸ·гҖҒз©әзҷҪзӯү

ps: texstudioзҡ„жҸҗзӨәгҖҒиӯҰе‘ҠгҖҒй”ҷиҜҜдҝЎжҒҜйқһеёёжҳҺзЎ®пјҢеҮәзҺ°й—®йўҳе…ҲзңӢжҸҗзӨәгҖӮ

2. ctex

иҝҷдёӘе…¶е®һжҳҜ miktex + winedt + дёӯж–Үж”ҜжҢҒ зҡ„йӣҶжҲҗзүҲпјҢз”Ёдёӯж–ҮеҶҷдҪңж—¶еҸҜзӣҙжҺҘз”ЁгҖӮ

дҪҶжҳҜжңҖеҗҺдёҖж¬Ўжӣҙж–°жҳҜеңЁ2016е№ҙ4жңҲпјҢеқ‘жҜ”иҫғеӨҡгҖӮ

  • ctex е®үиЈ…пјҲе®Ңж•ҙзүҲе®үиЈ…еҢ… ~1.3G е·ҰеҸіпјүпјҡ http://www.ctex.org/CTeXDownload

3. miktex + texstudio / (vscode + latex workshop)

иҝҷжҳҜйҖӮеҗҲжҠҳи…ҫзҡ„дҪҝз”Ёж–№жі•гҖӮ

  • дҪҝз”Ё Scoop е®үиЈ… miktexпјҢ texstudioпјҢ perl
  • еңЁ vscode йҮҢ е®үиЈ… latex workshop жҸ’д»¶пјҢеҸҜе®һж—¶йў„и§Ҳ
  • иҝҷжҳҜжҜ”иҫғжҠҳи…ҫзҡ„еҒҡжі•пјҢ miktexдёӯеҸӘжңүеҹәжң¬зҡ„еҢ…пјӣ еҗҺз»ӯз”ЁеҲ°йңҖиҰҒзҡ„еҢ…пјҢдјҡиҮӘеҠЁжҸҗзӨәдёӢиҪҪ
  • ж…ўж…ўжҠҳи…ҫеҗ§

latex е…Ҙй—Ёж•ҷзЁӢ

  • гҖҠLATEX жҺ’зүҲеӯҰд№ з¬”и®°гҖӢ
    • http://static.latexstudio.net/wp-content/uploads/2014/02/latexlog-1310.pdf
  • гҖҠдёҖд»ҪпјҲдёҚеӨӘпјүз®Җзҹӯзҡ„ LATEX д»Ӣз»ҚгҖӢ
    • http://mirrors.sjtug.sjtu.edu.cn/ctan/info/lshort/chinese/lshort-zh-cn.pdf

latex з®ҖеҚ•з”Ёжі•

  • еӣҫзүҮиҪ¬жҚў

    • pngиҪ¬eps: texliveдёӯ иҮӘеёҰдәҶдёҖдёӘе‘Ҫд»Өпјҡ bmeps
    • python и°ғз”Ё matplotlib иғҪзӣҙжҺҘеӯҳдёә eps ж јејҸпјҢеҸҜзӣҙжҺҘеӨҡдҝқеӯҳдёҖд»Ҫepsзҡ„

  • PPTиҪ¬дёәpngгҖҒeps

    • pptеҜјеҮәpng: й»ҳи®ӨdpiиҫғдҪҺпјҢйңҖиҰҒеңЁжіЁеҶҢиЎЁж”№дёҖдёӢdpi=300гҖӮ
    • pptеҜјеҮәeps: е…ҲеҸҰеӯҳдёәpdfпјҢеҶҚз”Ё adobe acrobat иҪ¬дёәepsгҖӮ
