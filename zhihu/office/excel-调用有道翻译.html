有道翻译 API 简介

  • http://fanyi.youdao.com/translate?&i="text"&doctype=xml

参数说明：

  • i="text"，引号内为要翻译的单词或句子，中文、英文都行，默认是中英互译；
  • doctype=xml，指定返回值的格式为 xml 格式，还可以是 text， json等

其他可选：

  • type，默认是type=auto；可以指定语言,比如中文->日语：type=zh_cn2ja
  • version，指定 api 版本等
