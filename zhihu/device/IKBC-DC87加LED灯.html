# IKBC DC87 加LED灯

title: IKBC DC87 加LED灯
date: 2020-7-24 22:20:22
tags: ikbc
categories: mechatronics

准备

  • LED灯，3mm无边LED灯 暖白色
  • 背光固件：DC87出厂默认没有背光，需要刷固件，关键词搜”信必成m20双模”

焊灯

DC87的灯位最上面一排没有预留焊锡，其他的预留了焊锡。

可以先加热后直接把led戳进去；或者用吸锡器先把预留的焊锡全部吸掉。

我这里直接加热后戳进去的，个别的需要补一点焊锡防止虚焊。

刷固件

  • 运行界面

  • 背光快捷键

快捷键 功能 备注
FN + HOME 打开/切换灯光效果
FN + END 关闭灯光效果
FN + ↑ 增加亮度
FN + ↓ 减少亮度
FN + INS 增加亮度
FN + DEL 减少亮度
FN + ← 灯光方向向左 灯光向左速度
FN + → 灯光方向向右 灯光向右速度
FN + PGUP 进入/退出自定义灯光 FN键灯亮
  • PS：默认的蓝牙快捷键
快捷键 功能 备注
FN + F1~F4 切换蓝牙设备1~4 ScrLk灯闪烁
FN + F5 开启蓝牙 空格左灯闪烁
FN + PrtSc 强制切换到蓝牙模式 ScrLk灯闪烁
FN + ScrLk 强制切换到有线模式
FN + Pause 恢复出厂设置 长按5s，空格灯闪烁
