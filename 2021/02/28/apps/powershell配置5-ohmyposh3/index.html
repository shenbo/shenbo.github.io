powershell配置(5)-ohmyposh3

发表于 2021-02-28 分类于 apps 阅读次数：

1.安装

以scoop安装oh-my-posh3。

scoop install oh-my-posh3

2. 配置文件

新建、修改配置文件，运行:

if (!(Test-Path -Path $PROFILE )) { New-Item -Type File -Path $PROFILE -Force }
code $PROFILE

加入以下内容：


chcp 65001
Set-PSReadLineOption -PredictionSource History

Invoke-Expression (oh-my-posh --init --shell pwsh --config "$(scoop prefix oh-my-posh3)/themes/ys.omp.json")

ref: (https://ohmyposh.dev/docs/installation)[https://ohmyposh.dev/docs/installation]

3. 自定义ys主题

  • 打开主题文件，路径： C:\Users\XXX\scoop\persist\oh-my-posh3\themes\ys.omp.json
{
  "final_space": true,
  "console_title": true,
  "console_title_style": "template",
  "console_title_template": "{{if .Root}}{{.User}}{{end}}@{{.Folder}} :: {{if contains \"powershell\" .Shell}}PS{{end}}",
  "blocks": [
    {
      "type": "prompt",
      "alignment": "left",
      "segments": [
        {
          "type": "text",
          "style": "plain",
          "foreground": "blue",
          "properties": {
            "prefix": "",
            "text": "#"
          }
        },
        {
          "type": "root",
          "style": "plain",
          "foreground": "red",
          "properties": {
            "root_icon": "\b\b\b%"
          }
        },
        {
          "type": "session",
          "style": "plain",
          "properties": {
            "user_info_separator": "<white> @ </>",
            "prefix": "",
            "user_color": "cyan",
            "host_color": "green"
          }
        },
        {
          "type": "path",
          "style": "plain",
