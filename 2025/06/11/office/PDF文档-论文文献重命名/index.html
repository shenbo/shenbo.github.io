PDF论文文献重命名

发表于 2025-06-11 分类于 office 阅读次数： 本文字数： 623 阅读时长 ≈ 2 分钟

pdf-renamer 可以对已下载的论文 PDF 文件 进行重命名。

  • https://github.com/MicheleCotrufo/pdf-renamer

调用 pdf2bib 以及 pdf2doi 查找论文的 DOI、并获得 BibTeX 数据。

  • https://github.com/MicheleCotrufo/pdf2bib

2. pdfrenamer 使用方法

args args description
path pdf文件或文件夹的路径
-h –help 帮助
-s –decrease_verbose 默认输出全部搜索记录
使用此选项，设为不输出
-ro –readonly 默认将doi信息写入pdf的metadata信息中
使用此选项，设为不写入
-f FORMAT 设定新文件名的格式，默认为 “{YYYY} - {Jabbr} - {A3etal} - {T}”
{YYYY} 出版年Year of publication
{MM} 出版月 (数字 digits)
{DD} 出版日 (数字 digits)
{J} 期刊全名Journal
{Jabbr} 期刊缩写（如有）
{Aall} 全部作者姓氏 (逗号分隔)
{Aetal} 第一作者姓氏 ‘et al.’
{A3etal} 前三作者姓氏 ‘et al.’
{aAall} 全部作者姓名 (逗号分隔)
{aAetal} 第一作者姓名 ‘et al.’
{aA3etal} 前三作者姓名 ‘et al.’
{T} 论文标题
-sf –sub_folders 是否重命名子文件夹，默认：False
-max_length_authors MAX_LENGTH_AUTHORS 设定作者的最大字符长度，默认：80
-max_length_filename MAX_LENGTH_FILENAME 设定文件名最大字符长度，默认：250
-max_words_title MAX_WORDS_TITLE 设定标题的最大字符长度，默认：20
-case CASE 连接符：驼峰、下划线、减号、空格
CASE：[‘camel’, ‘snake’, ‘kebab’, ‘none’ (default)]
-add_abbreviation_file PATH_ABBREVIATION_FILE 期刊缩写
-fr –force_rename 默认跳过相同文件
使用此选项，设为重命名每一个文件
-sd –set_default 将现有参数设置为默认值
-install–right–click 加入 Windows 右键菜单
-uninstall–right–click 删除 Windows 右键菜单

3. pdf2bib 使用方法

args args description
path pdf文件或文件夹的路径
-h –help 帮助
-v –verbose 默认仅输出搜索结果
使用此选项，设为输出搜索过程记录
-nostore –no_store_identifier_metadata 默认将doi信息写入pdf的metadata信息中
使用此选项，设为不写入
-s FILENAME_BIBTEX 将 bibtex 信息写入 FILENAME_BIBTEX 文本文件.
–make_bibtex_file FILENAME_BIBTEX 将 bibtex 信息写入 FILENAME_BIBTEX 文本文件
-clip –save_bibtex_clipboard 将 bibtex 写入粘贴板
-install–right–click 加入 Windows 右键菜单
-uninstall–right–click 删除 Windows 右键菜单

4. 加入右键菜单。

Windows 操作系统下，可以将 pdf-renamer 添加到右键菜单作为快捷方式。
