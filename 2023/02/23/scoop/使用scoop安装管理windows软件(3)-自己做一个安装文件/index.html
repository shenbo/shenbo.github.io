使用 scoop 安装管理 windows 软件（3）：自己制作一个安装配置文件

发表于 2023-02-23 分类于 apps 阅读次数： 本文字数： 204 阅读时长 ≈ 1 分钟

ref: (https://github.com/ScoopInstaller/Scoop/wiki/App-Manifests)

尝试自己制作一个安装配置文件， 软件是 N_m3u8DL-RE。

参数说明：

  • url: 下载地址

  • hash: 可缺省。 要添加的话： aria2 下载后会有输出，复制过来。

  • bin: 可执行文件的相对路径，根目录是 ~\scoop\apps\XXXX\current\

  • checkver: 可缺省。

  • autoupdate: 可缺省。
    这两项原理也搞不清楚，看其他类似软件的配置文件做相对应修改。
